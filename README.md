1️⃣ Task : Sentiment Analysis for Product Reviews 📝

Objective: To build a model that automatically classifies customer feedback as Positive or Negative using NLP techniques.

Dataset: IMDB__Dataset.csv (Movie Reviews).

Algorithms: Multinomial Naive Bayes.

---------------------------------------------------------------------------------------------------------------------

Workflow:

Preprocessing: Tokenization and removal of stop words to clean the text data.
Vectorization: Used TF-IDF to convert text into numerical features for the model.
Visualization: Created bar charts to analyze the distribution of sentiments across the dataset.

---------------------------------------------------------------------------------------------------------------------

Outcome: The model provides a clear classification report (Precision, Recall, F1-Score) to measure accuracy.
